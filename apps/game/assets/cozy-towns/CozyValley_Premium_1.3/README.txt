To make the character sprites easily customisable, I have cut them up into different layers. This can be a little confusing to work with, so I've included a guide for both the assembly and the animations.
If you don't care about (more complex) customization, or just need some characters for testing, there's a folder with pre-assembled characters. Please note that weapons and tools (besides the watering can and torch) still need to be added on top of these seperately.
